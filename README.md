## Running
1. Clone the repo
2. `cd EXE-Bot`
3. `npm install`
4. Open `json/config.json` and edit as your liking
5. Create a file in `json/` called `.env` and add the next:
```
token=''

OsuKey=''

GoogleAppApiKey=''
GoogleCseID=''

db_url=''
``` 
5. Edit it with your desired configuration
6. `npm run starttsc`
